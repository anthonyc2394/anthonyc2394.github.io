Freedom Project

The "Freedom Project" for is a year long project that consists of constructing an innovation about a topic about which we are enthusiastic about. The goal of this project is to teach people how coding can be seen in everyday activities such as producing music. For my project, I created 3 songs using Earsketch, all of them of different lengths and genres. I worked on this project all year and it was completely worth. I got an honorable mention and got to show off my talent in music.
