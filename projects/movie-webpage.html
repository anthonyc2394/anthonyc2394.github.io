Movie Webpage Project

I was assigned to make a movie webpage create a website using html and css to display a movie webpage with my partner, Jennifer Li.

In the end, we made a horror movie webpage about an adopted little girl with dementia that tries to kill her new family.

Some challenges I faced during this project is setting the background image. I tried *{background-image: url("https://i.pinimg.com/originals/13/5b/80/135b8031023bc124c5ea10d9f226ed00.jpg");}

It worked, but it looked like the image kept being added for each element. I looked at the elements and the only element that wasn't inside another element was the html element, so I tried html{background-image: url("https://i.pinimg.com/originals/13/5b/80/135b8031023bc124c5ea10d9f226ed00.jpg");}.

Some takeaways are communication, collaboration, and teamwork.

My next steps are to make a sequal, add more details to the story, and experiment with filtering images.
